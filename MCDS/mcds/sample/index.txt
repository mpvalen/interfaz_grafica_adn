The folders

.\60Co
.\125I
.\137Cs
.\Selected_Ions

contain sample input and output files.  Output files in the .\60Co, .\125I
and .\137Cs folders were created with the MS Windows version of the MCDS.
Output files in the .\Selected_ions folder were created with the Linux 
version.  Input files for the MS Windows and Linux executables are the 
same.
